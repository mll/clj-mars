# clj-mars

Memory Array Redcode Simulator written in pure clojure. Supporting ICWS-88 standard.

## Installation

Download from http://example.com/FIXME.

## Usage

    $ java -jar clj-mars-1.0.2-standalone.jar warrior1.red warrior2.red
